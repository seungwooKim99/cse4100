Phase III: Run Processes in Background in Your Shell

- make로 컴파일 진행
- ./myshell로 execute
- cmd를 입력하여 shell 명령어 처리
- exit로 종료

- 명렁어 & 로 bg에서 실행 가능
- jobs 로 Job queue에 등록된 job 확인 가능
- fg %number 로 number에 해당하는 job을 fg로 실행
- bg %number 로 number에 해당하는 job을 bg로 실행
- kill %number 로 number에 해당하는 job을 kill
